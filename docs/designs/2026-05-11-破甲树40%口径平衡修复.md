# 破甲树 40% 口径平衡修复

**日期：** 2026-05-11  
**文档定位：** 破甲系单轮数值削弱补充口径  
**适用范围：** 仅覆盖 `armor` 天赋树中的破甲供给与破甲转伤，不调整装备、不调整其他流派  

## 一、调整目标

本轮修复只解决一个问题：破甲树自身提供的破甲与转伤过高，导致“天赋自己就能堆出接近满破，再顺带转出过高全伤”。

最终目标如下：

- 不新增全局破甲封顶规则
- 纯天赋满配时，破甲树自身最多只提供 `40%` 护甲穿透
- `armor_pen_convert` 继续保留，但转伤倍率下调
- 装备额外提供多少破甲，不在本轮限制
- 崩塌、自动打击、审判日、首领猎杀、重甲特攻等机制不改

## 二、本轮正式口径

### 2.1 纯天赋破甲总量

破甲树纯天赋满配时，护甲穿透来源固定为：

- `armor_core`：Lv1~Lv5 = `12% / 14% / 16% / 18% / 20%`
- `armor_pen_up`：Lv1~Lv5 = `4% / 6% / 8% / 10% / 12%`
- `armor_filler_t1b`：Lv1~Lv5 = `1% / 2% / 3% / 4% / 5%`
- Armor Tier2 层满奖励：`+3%` 护甲穿透

满配合计：

```text
20% + 12% + 5% + 3% = 40%
```

这意味着“纯天赋最多 40%”是天赋供给上限，而不是运行时统一封顶。

### 2.2 破甲转全伤

`armor_pen_convert` 新口径：

- Lv1~Lv5 = `10% / 15% / 20% / 25% / 30%`

因此纯天赋满配时，满级转伤上限为：

```text
40% × 30% = 12% 全伤
```

如果后续装备继续提供额外破甲，那么转伤会继续提高；这部分属于装备数值设计范围，不在本轮拦截。

### 2.3 层满奖励变更

本轮只改破甲树 Tier2 层满奖励：

- 旧值：`护甲穿透 +15%`
- 新值：`护甲穿透 +3%`

其他层满奖励不动：

- Tier0：`全伤害 +10%`
- Tier1：`全伤害 +10%`
- Tier3：`崩塌易伤 +15%`
- Tier4：`审判日倍率 +1.0`

## 三、本轮明确不改的内容

以下内容全部保持原样：

- `armor_core` 的崩塌触发次数曲线
- 崩塌持续时间
- `armor_collapse_ext`
- `armor_auto_strike`
- `armor_ruin`
- `armor_ultimate`
- `armor_boss_hunter`
- `armor_heavy_atk`
- 其他三系全部节点
- 装备破甲词条与装备破甲上限

## 五、验证结果

本轮已补充并通过以下校验：

- 纯天赋满配护甲树时，实际护甲穿透为 `40%`
- 满级 `armor_pen_convert` 时，纯天赋基线只额外转出 `12%` 全伤
- 崩塌、自动打击、审判日相关后端测试未回归
- 天赋状态接口的动态效果描述与前端层满奖励展示同步通过
