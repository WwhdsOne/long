# 致命洞察（crit）工作原理

## 核心机制：暴击 → 死兆堆叠 → 收割循环

本篇系围绕"暴击击中弱点 → 获得死兆层数 → 不同阈值触发不同收割效果"展开。

### 死兆（Omen）系统

死兆是本系的核心资源。弱点暴击获得 1 层。每层死兆提供暴伤加成。不同层数阈值触发不同效果：

- **30 层**：死兆收割（2x 必暴伤害）
- **50 层**：死亡狂喜（消耗 50 层，6 秒暴伤 +200%，全攻击视为弱点）
- **100 层**（末日审判）：标记 2 个部位，击碎首个 → 100 死兆 + 暴伤 ×3，再击碎 → ×6
