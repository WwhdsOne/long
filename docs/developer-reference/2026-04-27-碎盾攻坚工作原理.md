# 碎盾攻坚（armor）工作原理

## 核心机制：穿甲 → 崩塌 → 部位削除

本篇系围绕"对重甲累计命中 → 触发崩塌（护甲归零）→ 崩塌状态内增伤 → 审判日削除 50% 最大生命"展开。

攻击重甲部位时累积点击计数。满 100 次触发崩塌（该部位护甲归零 8 秒 / 15 秒）。崩塌期间废墟打击增伤 +100%，审判日可在 100 次命中后直接削除重甲 50% 最大生命。

---

## 可调参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `collapseTrigger` | 100 | 触发崩塌所需重甲命中次数 |
| `collapseDuration` | 8 | 崩塌持续秒数 |
| `collapseDurationExtended` | 15 | 学习崩塌延长后 |
| `auto_strike.interval` | 20 | 自动打击间隔秒数 |
| `auto_strike.damageRatio` | 3.0 | 自动打击倍率 |
| `judgment_day.triggerCount` | 100 | 审判日触发所需命中次数 |
| `judgment_day.hpCutPercent` | 0.50 | 削除最大生命比例 |
