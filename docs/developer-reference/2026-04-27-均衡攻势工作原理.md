# 均衡攻势（normal）工作原理

## 核心机制：暴风连击 → 追击爆发

本篇系围绕"连击计数 → 追击爆发"的循环展开。每次点击增加连击计数，满 100 次触发一次追击爆发（一次性额外伤害），同时蓄力返还保留 30% 连击进度。

### 伤害公式

- **点击伤害** = 基础伤害 × (1 + 攻击力加成) × (1 + 全伤害加成) × 部位系数 × 其他
- **追击爆发** = 本次伤害 × 追击倍率 × 追击段数

```
追击爆发伤害 = baseDamage × chaseRatio × extraHits
触发时机：effectiveClicks % triggerCount == 0
触发后：effectiveClicks = clickCount × 0.3（蓄力返还）
```
